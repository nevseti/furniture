**Тема:** Стилизация веб-страницы с использованием современных подходов CSS


---

## Описание проекта

Проект представляет собой статическую веб-страницу для мебельного бренда "Avion", сверстанную по макету. Сайт демонстрирует каталог мебели, керамики и предметов интерьера. 
В текущей версии проведен рефакторинг кода по методологии **БЭМ**, добавлены интерактивные состояния элементов и плавная навигация.


---

## Используемые технологии

- **HTML5** — семантическая разметка
- **CSS3** — стилизация элементов
- **БЭМ** — методология именования классов
- **Git** — контроль версий
- **GitHub Pages** — хостинг проекта

## Семантические блоки страницы

1. **`<header>`** — шапка сайта с логотипом, навигацией и иконками пользователя
2. **`<section class="hero">`** — главный экран с приветствием и кнопкой коллекции
3. **`<section class="features">`** — блок преимуществ бренда
4. **`<section class="products">`** — каталог новых поступлений (керамика)
5. **`<section class="products products--popular">`** — популярные товары
6. **`<section class="newsletter">`** — форма подписки на рассылку
7. **`<section class="brand-story">`** — история бренда
8. **`<footer>`** — подвал сайта с меню, категориями и соцсетями
