# Верстка сайта по макету на HTML

**Ссылка на github pages: [https://nevseti.github.io/furniture/](https://nevseti.github.io/furniture/)**
**Практическое занятие:** 2  
**Тема:** Верстка сайта по макету (вариант 2) на HTML

---

## Описание проекта

Проект представляет собой статическую веб-страницу для мебельного бренда "Avion", сверстанную по макету. Сайт демонстрирует каталог мебели, керамики и предметов интерьера. Использована семантическая HTML-разметка и CSS стилизация.

---

## Используемые технологии

- **HTML5** — семантическая разметка
- **CSS3** — стилизация элементов
- **Git** — контроль версий
- **GitHub Pages** — хостинг проекта

## Семантические блоки страницы

1. **`<header>`** — шапка сайта с логотипом, навигацией и иконками пользователя
2. **`<section class="hero">`** — главный экран с приветствием и кнопкой коллекции
3. **`<section class="features">`** — блок преимуществ бренда
4. **`<section class="products-section">`** — каталог новых поступлений (керамика)
5. **`<section class="popular-products">`** — популярные товары
6. **`<section class="newsletter">`** — форма подписки на рассылку
7. **`<section class="brand-story">`** — история бренда
8. **`<footer>`** — подвал сайта с меню, категориями и соцсетями
